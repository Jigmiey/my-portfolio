Projects

3D Object Detection with Multimodal Fusion

This project focused on improving 3D object detection by combining camera images and LiDAR point clouds. Extensive testing was conducted using detectors like Cascade RCNN and Voxel-RCNN on the KITTI dataset.

Sentiment Analysis of Social Media Posts on Russia-Ukraine War

This project focuses on analyzing public sentiment toward Ukraine in the context of the ongoing Russia-Ukraine war using social media posts. By scraping tweets with relevant hashtags such as #UkraineWar and #RussiaWar, the data is cleaned and processed to prepare it for sentiment analysis. Several machine learning models, including Logistic Regression, Decision Trees, SVM, Random Forest, and XGBoost, are used to classify the sentiment of the tweets, with pretrained models like AWS and Bard also applied to enhance accuracy.

Motion Planning with Deep Reinforcement Learning

Implemented PPO, SAC, DQN, and DDPG algorithms for autonomous navigation in a corridor environment using a quadcopter. Environments were created using Unreal Engine and Microsoft AirSim.
